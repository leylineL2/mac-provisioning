# mac-provisioning

My mac-provisoning by Homebrew and Ansible.

## Requirements
- homebrew
- git
- python
- pip
- ansible

## Usage
```
$ git clone https://github.com/rednes/mac-provisioning.git
$ cd mac-provisioning
$ ansible-playbook site.yml -vv
```
